# TP 5 React : Gestion de l'état et structuration

Ce projet a été réalisé dans le cadre du TP 5 de React. L'objectif principal est de mettre en pratique la gestion des états (locaux et globaux) et la structuration des composants.

## Fonctionnalités abordées

1. **Formulaires React** : 
   - Création d'un formulaire contrôlé (gestion via le state avec `useState`).
   - Création d'un formulaire non-contrôlé (gestion via les références avec `useRef`).
2. **Lifting State Up** : Partage de l'état entre un composant parent et un enfant à travers un simulateur de saisie de température.
3. **Context API** : Utilisation de `useContext` pour gérer un état global (connexion et déconnexion d'un profil utilisateur).
